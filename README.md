# scheduler

> Scheduler app (Node )

## Requirements
> Node 8.0+

> Run Mongo server

## Build Setup

``` bash

Install dependencies
$ npm install

Serve with hot reload at localhost:3000
$ npm run dev

Build for production and launch server
$ npm run build
$ npm start

```

## Docker
``` bash
$ docker-compose up --build

Next time, you can just use
$ docker-compose up

```
